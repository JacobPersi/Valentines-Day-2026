# Valeintne's DayDay App 2026

A personal, interactive world map designed for couples to track their shared adventures and future travel dreams. This application serves as a digital scrapbook, allowing you to pin locations, add photos, and document memories from your journey together.

## Features

-   **Interactive World Map**: A beautiful, zoomable 3D globe (or map projection) to visualize your travels.
-   **Memory Dropping**: Click anywhere on the map to drop a pin.
-   **Categorization**:
    -   **Memories**: Places you've already visited.
    -   **Dreams**: Bucket list destinations you plan to visit.
-   **Personalized Legend**: Pins are color-coded by owner:
    -   **Shared**: Adventures taken together.
    -   **User 1 & User 2**: Individual trips or wishlists.
-   **Rich Media**: Attach photos and descriptions to every pin.
-   **Private & Local**: All data is stored securely in your browser's local storage (IndexedDB). No external servers or accounts required.
-   **Data Sync**: Easy JSON export/import allows you to back up your data or share it between devices.
-   **Customizable**: Personalize the site title, subtitle, and user names directly from the settings menu.
-   **Responsive Design**: optimized for both desktop and mobile viewing.

## Technology Stack

Built with modern web technologies for a smooth and responsive experience:

-   **React** & **TypeScript**
-   **Vite** for fast build tooling
-   **D3.js** for map rendering
-   **Tailwind CSS** for styling
-   **IndexedDB** for client-side data persistence
